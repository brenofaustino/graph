Detalhes de Desenvolvimento

linguagens utilizadas nesse site
Javascript HTML CSS

O planejamento para a criação de um site é indispensável, além disso, técnicas e padrões de desenvolvimento precisam ser utilizados de forma coesa para que a organização aliada à manutenibilidade seja preservada. Para o desenvolvimento dessa página, foram utilizados, portanto, o padrão BEM de nomenclatura CSS, o padrão MVC para organização dos arquivos e estrutura de código Javascript, a preservação da semântica HTML e os conceitos de Programação Orientada a Objetos para desenvolvimento de código.
